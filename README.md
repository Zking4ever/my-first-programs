# Early experimental Projects

## 1️⃣ C++ Bank System
A simple command-line banking system written in C++.

Features:
- Create new accounts
- Search for existing accounts
- Save account data into `.txt` files
- Basic file handling using C++ file streams

Concepts learned:
- File handling
- Data storage
- Simple system design
- User input handling


## 2️⃣ First Website UI
My first static website created using HTML and CSS.

Pages included:
- Home
- About
- Sign In
- Sign Up

Focus:
- UI layout
- Page navigation
- Styling
- Basic web structure

This project helped me understand how websites are structured and styled.

###  First PHP + MySQL Project
My first backend experiment connecting a website to a database. After creating the static site i connected it to a database to make it fullt functional

Features:
- PHP connection to MySQL
- Register a user
- Store user data in database

Concepts learned:
- Server-side programming
- Database connections
- Basic authentication logic


## 3️⃣ C++ Website Generator (Experimental Tool)

One of my most fun early experiments.

I attempted to **automate website creation using C++**.

The program runs in the terminal and asks the user questions.

Based on the answers, the program **generates HTML/CSS code automatically**.

This was an early exploration into **code generation and development automation**.
